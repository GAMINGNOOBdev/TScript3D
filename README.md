# TScript3D
3D rendering engine based on the concept of opengl and glm.

## Work in progress parts
- Texture mapping

## Documentation
Look at [Docs.md](https://github.com/GAMINGNOOBdev/TScript3D/blob/main/Docs.md) to see and read about the current api.

## How to use
In the standard tscript ide, click on "upload document" and upload the tscript3d.tscript file to be able to include it in your projects.

See the [example.tscript](https://github.com/GAMINGNOOBdev/TScript3D/blob/main/example.tscript) file.
